Drone Tracker

Drone tracker pulls in data made available by the dronestre.am API and plots it to a google map. Users can select a year and see all of the strikes in that year. The infowindow contains relevant info about the strike, as well as a link to a related news article.

Technologies

  • Google Maps API Integration
  • JQuery
  • AJAX
